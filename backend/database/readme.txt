How to setup database
1. Open xampp then start Apache and MySQL
2. Open phpMyAdmin -- http://localhost/phpmyadmin/
3. click new (in left side)
4. type "acm"(without quotes) in Database name then click create
5. You should see your newly created database named acm in the left side.
6. click acm
7. click import
8. click browse...
9. navigate to your acm-test-master/backend/database folder
10. you should see acm.sql
11. double-click acm.sql
12. click Go in phpMyAdmin

13. Congrats! you have now imported the database and you may now login.
